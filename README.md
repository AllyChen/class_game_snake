# 經典復古貪吃蛇

這是一個使用 React + Vite + Tailwind CSS 建立的簡潔貪吃蛇遊戲。

## 特色

- 經典貪吃蛇遊戲玩法
- 即時 Canvas 畫面渲染
- 鍵盤控制
- 分數與最佳成績記錄

## 本地執行

**先決條件：** Node.js

1. 安裝相依套件：
   `npm install`
2. 執行開發伺服器：
   `npm run dev`
3. 預覽產生結果：
   `npm run preview`
4. 建置專案：
   `npm run build`

## GitHub Pages 部署

此專案已設定 GitHub Actions 自動部署到 GitHub Pages。當 `main` 分支有更新時，工作流程會自動建置並發佈 `dist` 內容到 `gh-pages` 分支。
